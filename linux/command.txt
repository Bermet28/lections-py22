/ - символ корневой корневой директории(папки)


.. - обознаечает предыдущую директорию
. - обозночает текущую директорию
торий
1. ls(list) -> Выводит список всех директорий и файлов в текущей директории(рабочая директория)
ls - a -> Выводит список как и скрытых так и обычных файлов и директорий
ls-t -> Выводит список как и скрытых так и обычных файлов с подробной информацией 
2. cd(change directory) -> менят текущую директорию на укзаную
cd <название директории>

3. mkdir(make directory) -> Создает указанную директорию
mkdir <директория>

4. touch -> создает указаный файл
touch <name_of_file>.<extension(расширение(txt, docs))>

5. pwd(print working directory) -> выводит путь до текущей директории
 rmdir( remove directory) - удаляет указанную директорию ( не может удалить не пустую директорию
 rm -> удаление файлов
 rm -rf -> принудительное удаление файлов либо директорий
 mv- перемещает либо переименовывает файл либо директориюв зависимости от указанного второго аргумента
 mv <файл директория>и
 cat -> выводит содержимое указанного файла  
 nano -открывает графический интерфейс  в терминале для редакт
ирования файла

 echo <текст для записи> название файла
